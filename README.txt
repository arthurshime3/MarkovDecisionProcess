Download github repository: https://github.com/arthurshime3/MarkovDecisionProcess

To run MDPs:
1. Open MDP folder
2. Run rl_sim.jar
3. Select the algorithm you wish to run (value iteration, policy iteration, Q learning)
4. Load the map you wish to run (Under 'Mazes': Small Sample or Big Sample)
5. Click initialize
6. Click execute
